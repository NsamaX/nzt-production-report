This web application is designed to manage and report on the production line operations for Nozomi Company. It provides comprehensive functionalities to monitor and analyze manufacturing output.
Key Features
 * Production Quantity Reporting: Generates detailed reports on the quantity of various models produced within the production line.
 * PDF Report Generation: Allows for the creation of professional PDF reports for easy sharing and archival.
 * Excel Data Export: Supports exporting production data to Excel format for further analysis and integration with other systems.
Purpose
The primary purpose of this application is to streamline the reporting process for production metrics, ensuring accurate and accessible data for performance monitoring and operational decision-making.
